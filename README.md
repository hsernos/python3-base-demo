# Python3的基础学习案例
  `不兼容python2`

## 目录
```
├─base                  包括编码，输入、输出函数，数据类型和控制流结构
├─function              包括函数基础，Lombda表达式，闭包和装饰器
├─class                 包括类的基础和高阶用法
├─advanced_features     包括迭代器，生成器和上下文管理器
├─io                    包括读写文件和常见的文件和目录操作
├─exception             异常处理
├─regex                 包括正则表达式
├─process_thread        包括进程和线程
```
